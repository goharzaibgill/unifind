## What is UniFind Pakistan?

UniFind Pakistan is a free, comprehensive web platform helping Pakistani 
students and parents discover, compare, and apply to universities across 
Pakistan — all in one place, completely free.

Built because when I was applying to university, my mother had to research 
everything alone. No student should face that problem again.

---

## Features

- 🏫 **72+ Universities** across 26+ cities with HEC rankings, fees, programs
- ⚖️ **Side-by-side Comparison** of up to 4 universities
- 🧮 **Merit Calculator** — HEC aggregate formula (Matric + FSc + Entry Test)
- 🎓 **50+ Scholarships** with direct application links
- 📅 **Admission Tracker** — 430+ deadlines across the academic calendar
- 🔍 **Real-time Search** with live suggestions and multi-filter system
- 📄 **PDF Export** for comparison and merit results
- 🌙 **Dark/Light Theme** toggle with persistence
- 📱 **Fully Responsive** — mobile, tablet, desktop
- ✅ **Full SEO** — structured data, sitemap, meta tags

## Tech Stack

| Technology | Purpose |
|-----------|---------|
| HTML5 | Page structure |
| CSS3 | Styling, animations, dark/light theme |
| Vanilla JavaScript | All functionality, no frameworks |
| Google Fonts (Poppins) | Typography |
| localStorage | Theme + compare list persistence |
| jsPDF | PDF export feature |

No React, no npm, no build tools. Pure HTML + CSS + JS.
